我的卧室之旅

发表于 2021-05-14 更新于 2021-12-29 分类于 literature

塞维尔·德·梅伊斯特因为决斗而被禁足42天，期间他突发奇想，穿着一身红蓝相间的睡衣在自己的卧室里旅行。先是出版了一本《我的卧室之旅》，然后写了一本《卧室夜游》。可惜的是这两本书最后都跑题了。

因为阿兰·德波顿的《旅行的艺术》里赞誉了我喜欢的爱德华·霍普（我在大概一周前提到过）以及波德莱尔，我写了一篇《岳阳游记》。这篇是卧室游记，尽管我根本没有真正的卧室，它看上去那么的狭窄简陋，布满了我无法预料的灯光和鼾声。但我依旧要写这一篇文章，不是我像塞维尔·德·梅伊斯特那样立志成为一位室内旅行家，仅仅是因为我没有一部相机，也没有足够牢固的记忆确保我不会忘记我当下的处境。尽管如此，这篇文章也不会像左拉的那样，用镁制的手术刀那样准确地捕捉每一寸光线，否则我必须详尽地表述我室友超量的鞋是如何诡异而真实地塞进那不大的鞋架里；也不能像表现主义者那样做，因为我的确不清楚蟑螂和蜘蛛的巢究竟存在于哪个角落。我只是描述在我窥探下的卧室的细节，因此，我首先要向我的椅子致敬。

外太空游记<6>:四大恶星

发表于 2020-05-04 更新于 2021-12-29 分类于 literature

四大恶星指的是艾欧、欧罗巴、加尼美德和卡里斯托，它们以从近到远的顺序围绕在木星周围；说它们是四大恶星，不仅仅因为它们长得穷凶极恶，更因为，它们自作主张地自诩为希腊神话中的美人。看到它们你这辈子都不会想吃冰激凌球了。

四大恶星的老大是艾欧，它个头不是最大，不过长得最为凶恶。它比月球稍微大一点，而就是在这样的脸上，长满了密密麻麻的、向外突出的青黄色的痈，淋漓的血胡乱地涂抹在皮肤上。这些痈是火山，艾欧一直不断地处于愤怒之中，这使它的体温达到了2000多度（比金星要热得多！）。它的大气中布满了各类硫类气体，从痈疮中也不断涌出富含硫化物的岩浆。艾欧星人体内依靠硫化还原反应为身体功能，甚至，他们用硫化物组成自身，他们吃火山岩浆，喝CS2，这样他们的皮肤就不可避免地染上黄色，他们看上去就像被黄色颜料覆盖的地球人。
